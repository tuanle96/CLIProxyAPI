🔍 Useful Queries

Total usage today:

docker exec cliproxy-postgres psql -U cliproxy -d cliproxy -c "
SELECT
  COUNT(*) as requests,
  SUM(input_tokens) as input_tokens,
  SUM(output_tokens) as output_tokens,
  SUM(cost_usd) as total_cost_usd
FROM usage_events
WHERE timestamp >= CURRENT_DATE;"

Top models by usage:

docker exec cliproxy-postgres psql -U cliproxy -d cliproxy -c "
SELECT
  model,
  COUNT(*) as requests,
  SUM(total_tokens) as tokens
FROM usage_events
GROUP BY model
ORDER BY requests DESC
LIMIT 10;"

Cost breakdown by provider:

docker exec cliproxy-postgres psql -U cliproxy -d cliproxy -c "
SELECT
  provider,
  COUNT(*) as requests,
  SUM(cost_usd) as total_cost
FROM usage_events
GROUP BY provider
ORDER BY total_cost DESC;"

🛠️ Management

Docker Commands:

# Check container status
docker ps | grep cliproxy-postgres

# View logs
docker logs cliproxy-postgres -f

# Connect to database
docker exec -it cliproxy-postgres psql -U cliproxy -d cliproxy

# Backup database
docker exec cliproxy-postgres pg_dump -U cliproxy cliproxy > backup.sql

# Stop container
docker compose stop postgres

# Start container
docker compose start postgres
